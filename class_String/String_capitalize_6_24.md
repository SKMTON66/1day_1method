- [String\#capitalize \(Ruby 4\.0 リファレンスマニュアル\)](https://docs.ruby-lang.org/ja/latest/method/String/i/capitalize.html) <br>
capitalize(*options) -> String

文字列先頭の文字を大文字に、残りを小文字に変更した文字列を返すメソッド。

```rb
p "apple".capitalize #=> "Apple"
p "apPlE".capitalize #=> "Apple"
p "APPLE".capitalize #=> "Apple"
```
