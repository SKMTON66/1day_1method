- [Array\#sort \(Ruby 4\.0 リファレンスマニュアル\)](https://docs.ruby-lang.org/ja/4.0/method/Array/i/sort.html) <br>
sort -> Array <br>
sort! -> self <br>
sort {|a, b| ... } -> Array <br>
sort! {|a, b| ... } -> self

全ての要素を昇順に並び替えた配列を返す。`sort!`は破壊的な変更を行う。<br>
ブロックを渡した場合、要素同士の比較をブロックを用いて行う。

```rb
ary = ["11", "10", "13", "12", "15", "14", "111", "112"]

p ary.sort #=> ["10", "11", "111", "112", "12", "13", "14", "15"] ：数値比較ではなく、文字列ソートなのでこのような出力結果になる
p ary #=> ["11", "10", "13", "12", "15", "14", "111", "112"]
p ary.sort! #=> ["10", "11", "111", "112", "12", "13", "14", "15"]
p ary #=> ["10", "11", "111", "112", "12", "13", "14", "15"] ：破壊的な変更が行われている

p ary.sort { |a, b| a.to_i <=> b.to_i } #=> ["10", "11", "12", "13", "14", "15", "111", "112"] ：数値に変換して比較したパターン
p ary.sort_by { it.to_i } #=> ["10", "11", "12", "13", "14", "15", "111", "112"] ：sort_byを使った書き方
```
