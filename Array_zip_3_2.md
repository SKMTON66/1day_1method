- [Array\#zip \(Ruby 4\.0 リファレンスマニュアル\)](https://docs.ruby-lang.org/ja/4.0/method/Array/i/zip.html) <br>
zip(*lists) -> [[object]] <br>
zip(*lists) {|v1, v2, ...| ...} -> nil

複数の配列から同じインデックスの要素を組み合わせて新しい配列を作るメソッド。

```rb
ary = %w(1 2 3)

p ary.zip(%w(4 5 6)) #=> [["1", "4"], ["2", "5"], ["3", "6"]]
p ary.zip(%w(4 5 6), %w(7 8 9)) #=> [["1", "4", "7"], ["2", "5", "8"], ["3", "6", "9"]]
ary.zip(%w(4 5 6), %w(7 8 9)) { p(it * 2) } 
#=> ["1", "4", "7", "1", "4", "7"] 
#=> ["2", "5", "8", "2", "5", "8"] 
#=> ["3", "6", "9", "3", "6", "9"] 
```
