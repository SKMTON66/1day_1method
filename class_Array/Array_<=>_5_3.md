- [Array\#<=> \(Ruby 4\.0 リファレンスマニュアル\)](https://docs.ruby-lang.org/ja/latest/method/Array/i/=3c=3d=3e.html) <br>
self <=> other -> -1 | 0 | 1 | nil

selfとotherの各要素を、それぞれ <=> で比較し、-1 | 0 | 1 | nil を返すメソッド。

```rb
p [1, 2, 3] <=> [1, 2, 3] #=> 0 # 全要素が等しく、長さも同じのため
p [1, 2, 3] <=> [1, 3, 2] #=> -1 # 2番目の要素が 2 < 3 のため
p [1, 2, 3] <=> [1, 2] #=> 1 # 最初の2要素だけ等しく、selfのほうが長いため
p [1, 2, 3] <=> [1, 2, 3, 4] #=> -1 # 最初の3要素だけ等しく、otherのほうが長いため
p [1, 2, 3] <=> ["A", "B", "C"] #=> nil # 比較不可能なため
```
